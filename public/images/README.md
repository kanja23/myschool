# MySchool — Public Images

Place your photos in this folder. The app will load them automatically.

## Required Images

| Filename | Description | Ideal Size |
|---|---|---|
| `hero-child.jpg` | Smiling child in school uniform — main hero photo | 800x1000px portrait |
| `hero-child-2.jpg` | Second child or classroom scene (optional) | 600x800px |
| `school-bg.jpg` | School building or classroom background (optional) | 1920x1080px |

## Where to Get Free Photos

1. **Unsplash.com** — search "African school child uniform smiling" — free commercial use
2. **Pexels.com** — same search, same rights
3. **Your own school** — photograph a child with parent permission — most authentic

## Photo Tips

- Bright natural light works best
- Child should be smiling, looking happy
- School uniform preferred (green, navy, or grey)
- Portrait/vertical orientation for the hero
- High resolution (at least 800x1000px)

## After Adding Photos

Just upload to this folder on GitHub. No code changes needed.
The hero section will automatically show `hero-child.jpg`.
